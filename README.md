# Predicción de fallo industrial

Proyecto de mantenimiento predictivo industrial basado en técnicas de aprendizaje supervisado.  
El objetivo es analizar los datos disponibles, identificar patrones relevantes y generar predicciones sobre posibles fallos en equipos industriales.

## Objetivos
- Realizar un análisis exploratorio de los datos.
- Preprocesar las variables relevantes.
- Entrenar y evaluar modelos supervisados.
- Comparar resultados y extraer conclusiones.

## Tecnologías utilizadas
- Python
- Jupyter Notebook
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn

## Estructura del proyecto
- `prediccion_fallo_industrial.ipynb`: notebook principal con el análisis completo.
- `train.csv`: conjunto de entrenamiento.
- `test.csv`: conjunto de prueba.
- `naive_bayes.py`: implementación auxiliar del modelo Naive Bayes.
- `decision_tree_c45.py`: implementación auxiliar del árbol de decisión C4.5.

## Proceso realizado
1. Exploración y limpieza de datos.
2. Análisis de variables.
3. Entrenamiento de modelos supervisados.
4. Evaluación de resultados.
5. Generación de predicciones finales.

## Resultados
Se compararon distintos enfoques de clasificación para predecir fallos industriales a partir de variables del sistema.  
El proyecto permite entender el flujo completo de un problema de mantenimiento predictivo, desde el análisis inicial hasta la obtención de predicciones.
